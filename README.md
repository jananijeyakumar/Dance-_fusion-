# 💃 Dance Fusion - Multi-Page Website

## 📌 Project Overview
Dance Fusion is a responsive multi-page dance academy website developed using HTML, CSS, and JavaScript. The website provides information about the academy, dance services, gallery, testimonials, contact details, and user authentication pages.

The project is designed with a modern UI and smooth navigation to provide a better user experience.

---

## 🚀 Features

- Responsive Design
- Modern User Interface
- Multi-page Navigation
- About the Academy
- Dance Services
- Gallery Section
- Student Testimonials
- Contact Page
- Login Page
- Signup Page
- Interactive JavaScript Features
- Mobile-Friendly Layout

## 📖 Pages Description

### 🏠 Home
Introduces the Dance Fusion Academy with attractive banners and navigation.

### 👨‍🏫 About
Provides information about the academy, vision, and mission.

### 💃 Services
Displays different dance styles and training programs offered.

### 🖼️ Gallery
Showcases dance performances and academy events.

### ⭐ Testimonials
Displays feedback and experiences shared by students.

### 📞 Contact
Contains academy contact details and inquiry form.

### 🔐 Login
Allows existing users to log in.

### 📝 Signup
Allows new users to create an account.
